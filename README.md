This crate contains functions for interacting with emulated or physical YM2149-adjacent PSG chips.
It provides a CommandOutput trait that allows the end user to define how register-value pairs
generated by the code are handled.

# Features
- **notes**: Basic musical note support
- **read**: I/O read operation support

### Examples
Examples located in the `examples/` directory display register write operations in the stdout - they require `std` to function and are intended for demonstration purposes only. For `no_std` examples, take a look in `examples/embedded/`

# Compatibility
This code should be backwards-compatible with other PSG sound chips based on the AY-3-8910 core, such as the AY-3-89xx series chips, the YM2149, AY-3-8910 and Sunsoft 5B.

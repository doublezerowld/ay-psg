This crate contains functions for interacting with emulated or physical YM2149-adjacent PSG chips.
It provides a CommandOutput trait that allows the end user to define how register-value pairs
generated by the code are handled.

# Features
- **notes**: Basic musical note support
- **read**: I/O read operation support

# Compatibility
This code should be backwards-compatible with other PSG sound chips based on the AY-3-8910, such as the YM2149, AY-3-8910 and Sunsoft 5B.
